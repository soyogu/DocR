DocRは、brms等の強力なR分析をExcel上で手軽に実行できるツールです。

「だれでも今日中にベイズ分析ユーザーになれる。」

データ分析の「最初の一歩」をスムーズに

  • 高度な統計モデルも直感的に実行。
  • Rのヘビーユーザーも納得の柔軟性。
  • エクセルファイルをその場ですぐに分析。

操作画面（Excelだけで完結します）

普段お使いのExcelのリボンに「DocR」のメニューが追加されます。変数を選んでクリックするだけで、裏側でRが走り、結果が新しいシートに出力されます。

搭載している分析機能一覧

  • Describe (psych) - 記述統計・分布プロット表示（Ver 1.5以降）
  • Correlation (cor) - 相関分析（ベイジアン相関・偏相関）（Ver 1.5以降）
  • Linear Regression (lm) - 単回帰・重回帰分析
  • GLM (glm) - 一般化線形モデル
  • ANOVA (afex) - 分散分析
  • Linear Mixed (lmer) - 線形混合モデル
  • Elastic Net (glmnet) - 正則化回帰（Lasso/Ridge）（Ver 1.5以降）
  • Bayesian (brms) - ベイズ階層モデル
  • BART (dbarts) - ベイジアン加法回帰木（非線形予測）（Ver 1.5以降）
  • Factor Analysis (fa/principal) - 探索的因子分析 / 主成分分析
  • SEM / CFA (lavaan) - 構造方程式モデリング / 確認的因子分析
  • LPA / LCA (tidyLPA/poLCA) - 潜在変数モデル
  • Text Mining（日本語専用） - 対応分析・TF-IDFスコア・出現頻度・共起ネットワーク・ワードクラウド
Mac対応しました！（しかし現在手元にMacが無いためMac版は旧バージョンで止まっています 。）
