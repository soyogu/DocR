分析機能一覧

目的の分析手法を選んでください。

  • Describe (psych) - 記述統計・分布プロット表示
  • Correlation (cor) - 相関分析（ベイジアン相関・偏相関）
  • Linear Regression (lm) - 単回帰・重回帰分析
  • GLM (glm) - 一般化線形モデル
  • ANOVA (afex) - 分散分析
  • Linear Mixed (lmer) - 線形混合モデル
  • Elastic Net (glmnet) - 正則化回帰（Lasso/Ridge）
  • Bayesian (brms) - ベイズ階層モデル
  • BART (dbarts) - ベイジアン加法回帰木（非線形予測）
  • Factor Analysis (fa/principal) - 探索的因子分析 / 主成分分析
  • SEM / CFA (lavaan) - 構造方程式モデリング / 確認的因子分析
  • LPA / LCA (tidyLPA/poLCA) - 潜在変数モデル
  • Text Mining（日本語専用） - 対応分析・TF-IDFスコア等

便利な使い方

  • 図の編集 - トリミング、色やフォントの変更など
