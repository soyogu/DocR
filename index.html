DocR

現在β版です。動作確認などご協力ください！なんとマルチリンガル！
まだWindowsだけですがMac対応もしたいです。

DocRの3つの特徴

1. Excelの操作感そのままに、Rの分析を実装

Excelの画面から直感的に変数を選択するだけで、裏側でRが計算を実行します。プログラミングの学習コストをゼロにし、本来の「データの解釈」に集中できます。

2. 最先端の統計手法（ベイズ推定からSEMまで）

標準のExcelでは不可能な、線形混合モデル（lmer）、探索的因子分析（fa）、構造方程式モデリング（SEM: lavaan）、さらにはbrmsを用いたベイズ階層モデルまで、学術研究の最前線で求められる強力なパッケージ群をワンクリックで呼び出せます。

3. 生成AIに尋ねやすい設計

モデル式や出力結果のサマリーはR準拠なので、そのまま生成AI（GeminiやChatGPTなど）に貼り付け、「出力結果の見方や論文への書き方をAIに相談する」という、分析の勉強にもなるワークフローを前提に設計されています。

操作画面（Excelだけで完結します）

普段お使いのExcelのリボンに「DocR」のメニューが追加されます。変数を選んでクリックするだけで、裏側でRが走り、結果が新しいシートに出力されます。

こんなあなたも・・・

  • 論文の査読で、いきなり線形混合モデル（lmer）を要求されて焦っている
  • ベイズ統計はアプリケーションのインストールで挫折した
  • 受け取ったExcelデータをその場でササっと眺めたい
  • スイスアーミーナイフのような便利なツールが欲しい！

搭載している分析機能一覧

  • Linear Regression (lm) - 単回帰・重回帰分析
  • GLM (glm) - 一般化線形モデル
  • ANOVA (afex) - 分散分析
  • Linear Mixed (lmer) - 線形混合モデル
  • Bayesian (brms) - ベイズ階層モデル
  • Factor Analysis (fa) - 探索的因子分析
  • SEM / CFA (lavaan) - 構造方程式モデリング / 確認的因子分析
  • LPA / LCA (tidyLPA/poLCA) - 潜在変数モデル
  • テキスト分析（日本語のみ） - 対応分析・TF-IDFスコア・出現頻度・共起ネットワーク・ワードクラウド

インストールガイド

はじめにDockerをインストールし、その後でこのDocRをインストールする順序になります。

ステップ0：Dockerのインストール

Docker Desktopのダウンロードページ

  • [Download Deocer Desktop]という白いボタンからダウンロード。
  • インストール中のチェックボックスはデフォルトのままでOKです。
  • 途中でユーザー登録を勧められる場面がありますが、「Skip」でOKです。

ステップ1：DocRのインストール

DocR_0_8.zip をダウンロード (638 MB)

  • 展開したフォルダ内にある Install_DocR.bat を実行します。
  • 「DocR.xlam has been successfully installed.」と表示されたら完了です。

ステップ2：Excelでのアドイン有効化

  1. Excelの「ファイル」＞「オプション」＞「アドイン」をクリック。
  2. 管理の「Excel アドイン」から「設定」ボタンを押し、「DocR」にチェックを入れます。

ユーザーマニュアル

基本仕様と注意事項

  • データ選択: 分析前にデータのセル範囲（最低2行×2列以上）を選択してください。
  • 変数名: 選択範囲の1行目はヘッダーとして処理されます。
  • データクレンジング: 空白セルや計算エラーが含まれていると実行されません。

AIを活用した分析・勉強のTIPS

分析の仕方も結果の読み方もAIに聞くとよくわかります。ぜひお試しください！

プロンプト例：分析手法の相談
参加者に3種類の異なる香り（無臭、フローラル、シトラス）を嗅がせ、魅力度を評価してもらいました。参加者は30名で、データは『参加者ID』『香り条件』『写真ID』『魅力度スコア』という変数名でExcelに入っています。 Rのlmerで解析したいのですが、固定効果と変量効果には何を指定すべきですか？
プロンプト例：出力結果の解釈
Rのbrmsでこんな結果が出ました。[結果をペースト] これはどういう意味ですか？サマリーの中にある『R-hat』って何ですか？
プロンプト例：作業を減らす
「・・・（略）・・・という構造を考えています。この構造をRのlavaanで検証するためのシンタックスを書いてください。
「 だれでも今日中にベイズ分析ユーザーになれる」
をコンセプトに作りました。

よくある質問（FAQ）

無料で使えますか？

はい、完全に無料でご利用いただけます。研究・教育の発展を目的としています。

複雑なモデルも扱えますか？

モデル式のテキストエリアを直接編集できるので上級者も思うままのモデルを扱えます。

Mac版はありますか？

現在はWindows版のみの提供となっています。Mac版の開発機材購入に向けたご支援を募集中です！

はじめに立ち上がる青いウィンドウは何ですか？

バックエンドに使われているDockerのウインドウです。立ち上がった後に閉じていただいても大丈夫です。
