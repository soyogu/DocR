DocR

手早い診断、深い洞察。

DocRは、brms等の強力なR分析をExcel上で手軽に実行できるツールです。分析初心者にもRのヘビーユーザーにも使いやすく設計されています。

データ分析の「最初の一歩」をスムーズに

  • 高度な統計モデルも直感的に実行できます。
  • 出力結果をAIに共有し、解釈や次の方針を相談する際のリファレンスになります。
  • DocRは手軽なのでデータ分析に慣れ親しむことができます。

多様なデータを、その場で手早く

  • 色々な人から受け取る様々なデータに対してその場で構造を把握できます。
  • 本格的な解析の前の、予備分析や現状把握にかかる手間を省きます。
  • Rのヘビーユーザーにも使いやすい柔軟性を持っています。

操作画面（Excelだけで完結します）

普段お使いのExcelのリボンに「DocR」のメニューが追加されます。変数を選んでクリックするだけで、裏側でRが走り、結果が新しいシートに出力されます。

搭載している分析機能一覧

  • Linear Regression (lm) - 単回帰・重回帰分析
  • GLM (glm) - 一般化線形モデル
  • ANOVA (afex) - 分散分析
  • Linear Mixed (lmer) - 線形混合モデル
  • Bayesian (brms) - ベイズ階層モデル
  • Factor Analysis (fa/principal) - 探索的因子分析 / 主成分分析
  • SEM / CFA (lavaan) - 構造方程式モデリング / 確認的因子分析
  • LPA / LCA (tidyLPA/poLCA) - 潜在変数モデル
  • テキスト分析（日本語のみ） - 対応分析・TF-IDFスコア・出現頻度・共起ネットワーク・ワードクラウド

インストールガイド

✔ はじめにDockerをインストールし、その後でこのDocRをインストールする順序になります。

✔ もしあなたがすでにRをインストールしていても、その環境には一切干渉しません。

ステップ0：Dockerのインストール

Docker Desktopのダウンロードページ

  • [Download Deocer Desktop]という白いボタンからダウンロード。
  • インストール中のチェックボックスはデフォルトのままでOKです。
  • 途中でユーザー登録を勧められる場面がありますが、「Skip」でOKです。

ステップ1：DocRのインストール

Windows版：DocR_Win_1_0_1.zip をダウンロード

Mac版：DocR_Mac_1_0_0.zip をダウンロード

  • 展開したフォルダ内にある Install_DocR.bat （MacではInstall_DocR_Mac.command）を実行します。
    Windowsで青い警告画面（PCが保護されました）が出た場合

    メッセージ内の「詳細情報」という文字をクリックし、右下に現れる「実行」ボタンを押してください。

    Macで「開発元を検証できない」と警告が出た場合

    そのままダブルクリックせず、ファイルを右クリック（または Controlキー を押しながらクリック）し、表示されたメニューから「開く」を選択して実行してください。

  • 「DocR.xlam has been successfully installed.」と表示されたら成功です。

ステップ2：Excelでのアドイン有効化

Windows
  1. Excelの「ファイル」＞「オプション」＞「アドイン」をクリック。
  2. 管理の「Excel アドイン」から「設定」ボタンを押し、「DocR」にチェックを入れます。
Mac

何もせずにメニューに表示されます。

使い方

基本仕様

  • 基本操作: データのセル範囲を選択し、分析のボタンを押してください。
  • 変数名: 選択範囲の1行目はヘッダーとして処理されます。「()」などが使われていても大丈夫です。
  • 式の編集: 分析によっては、モデル式を直接編集できるものもあります。
  • 結果出力: 結果出力は基本的にはRのsummaryです。一部追加や修正されている場合もあります。

AIを活用した分析・勉強のTIPS

分析の仕方も結果の読み方もAIに聞くとよくわかります。ぜひお試しください！

プロンプト例：分析手法の相談
参加者に3種類の異なる香り（無臭、フローラル、シトラス）を嗅がせ、魅力度を評価してもらいました。参加者は30名で、データは『参加者ID』『香り条件』『写真ID』『魅力度スコア』という変数名でExcelに入っています。 Rのlmerで解析したいのですが、固定効果と変量効果には何を指定すべきですか？
プロンプト例：出力結果の解釈
Rのbrmsでこんな結果が出ました。[結果をペースト] これはどういう意味ですか？サマリーの中にある『R-hat』って何ですか？
プロンプト例：作業を減らす
「・・・（略）・・・という構造を考えています。この構造をRのlavaanで検証するためのシンタックスを書いてください。
「 だれでも今日中にベイズ分析ユーザーになれる」
をコンセプトに作りました。

よくある質問（FAQ）

無料で使えますか？

はい、完全に無料でご利用いただけます。研究・教育の発展を目的としています。

複雑なモデルも扱えますか？

モデル式のテキストエリアを直接編集できるので上級者も思うままのモデルを扱えます。

Mac版はありますか？

はい。しかし人から借りたMacでなんとか開発したので、自分の作業用のMacを買わなければ・・・。

なぜDocRは既存のR環境と競合しないのですか？

DocRは、あなたのコンピュータのOSに直接Rをインストールするのではなく、「Dockerコンテナ」という独立した隔離空間の中でRを動かすからです。またそれゆえ、ライブラリの依存関係やバージョンの不一致に悩む心配もありません。

はじめに立ち上がる青いウィンドウは何ですか？

バックエンドに使われているDockerのウインドウです。立ち上がった後に閉じていただいても大丈夫です。
