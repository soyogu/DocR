3 Key Features of DocR

1. Run R Analyses with the Familiar Feel of Excel

Simply select variables intuitively from the Excel screen, and R runs the calculations in the background. It eliminates the learning curve of programming, allowing you to focus on the essential task: "interpreting your data."

2. Cutting-Edge Statistical Methods (From Bayesian Inference to SEM)

With just one click, you can access powerful R packages required at the forefront of academic research—such as Linear Mixed Models (lmer), Exploratory Factor Analysis (fa), Structural Equation Modeling (SEM: lavaan), and even Bayesian Hierarchical Models using brms—which are impossible in standard Excel.

3. Designed to be AI-Prompt Friendly

Model formulas and output summaries are R-standard, designed for a workflow where you can paste them directly into generative AI (like Gemini or ChatGPT) to ask, "How do I interpret these results?" or "How should I write this in my paper?" This makes it a great tool for learning statistics while conducting your analysis.

User Interface (Complete Everything within Excel)

A "DocR" menu is added to your familiar Excel ribbon. Just select variables and click; R runs in the background and outputs the results to a new sheet.

List of Available Analytical Features

  • Linear Regression (lm) - Simple / Multiple Linear Regression
  • GLM (glm) - Generalized Linear Models
  • ANOVA (afex) - Analysis of Variance
  • Linear Mixed (lmer) - Linear Mixed Models
  • Bayesian (brms) - Bayesian Hierarchical Models
  • Factor Analysis (fa) - Exploratory Factor Analysis
  • SEM / CFA (lavaan) - Structural Equation Modeling / Confirmatory Factor Analysis
  • LPA / LCA (tidyLPA/poLCA) - Latent Variable Models
  • Text Analysis (Japanese only) - Correspondence Analysis, TF-IDF, Word Frequency, Co-occurrence Network, Word Cloud

Installation Guide

✔ Please install Docker first, and then install the DocR add-in.

✔ Even if you already have R installed, DocR will not interfere with your existing environment.

Step 0: Install Docker

Docker Desktop Download Page

  • Download by clicking the white "[Download Docker Desktop]" button.
  • You can leave the checkboxes as default during the installation process.
  • You may be prompted to register a user account, but you can safely click "Skip".

Step 1: Install DocR

Download DocR_0_8.zip (638 MB)

  • Run the Install_DocR.bat file located inside the extracted folder.
  • The installation is complete when "DocR.xlam has been successfully installed." is displayed.

Step 2: Enable the Add-in in Excel

  1. In Excel, click "File" > "Options" > "Add-ins".
  2. Select "Excel Add-ins" from the Manage dropdown menu, click "Go...", and check the box for "DocR".

User Manual

Basic Specifications & Notes

  • Data Selection: Select the data cell range (minimum 2 rows x 2 columns) before running an analysis.
  • Variable Names: The first row of the selected range is automatically treated as the header (variable names).
  • Data Cleansing: The analysis will not run if the data contains blank cells or Excel calculation errors.

Frequently Asked Questions (FAQ)

Is it free to use?

Yes, it is completely free to use. Our goal is to contribute to the advancement of research and education worldwide.

Can it handle complex models?

Yes, advanced users can directly edit the model formula text area to run customized models as desired.

Is there a Mac version?

Currently, only the Windows version is available. We are looking for donations to purchase equipment to develop a macOS version!

Why doesn't DocR conflict with my existing R installation?

This is because DocR runs R within an isolated "Docker container" rather than installing it directly on your OS. Consequently, you never have to worry about library dependencies or version conflicts.

What is the blue window that appears upon startup?

It is the Docker terminal window used for the backend. You may safely close it once the application has fully launched.

How should I cite this software?
